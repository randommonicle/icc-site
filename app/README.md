# ICC Field App (placeholder)

The ICC field app (**D-012**) is a **concurrent, fully-integrated client** of the platform API — equal to the public website, not a deferred final phase. It is not designed or built yet.

This directory is a placeholder so the monorepo layout (**D-014**) exists from the start. The app's own build (Capacitor shell, UI) begins once the Phase 2 API exists to consume (`shared/contract/` + `server/`). Until then this is intentionally empty.
